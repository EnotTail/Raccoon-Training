# 🦝 Енотов Фитнес

Персональный PWA-трекер фитнеса и питания. Работает офлайн, сохраняет данные на устройстве.

## 📱 Установка

1. Откройте ссылку в Safari (iOS) или Chrome (Android)
2. Нажмите «Поделиться» → «На экран Домой»
3. Готово! Приложение появится как нативное

## ✨ Возможности

- 🦝 **Стартовый экран** — ввод пола, роста, веса
- 🎯 **Авто-расчёт целевого веса** по ИМТ
- 📊 **Динамический ИМТ** с визуальной шкалой
- 💧 **Трекер воды** — 8 стаканов в день
- 🏋️ **3 разных тренировки** — ноги, руки, смешанная
- 🍽️ **Разное питание** каждый день недели
- 📆 **Недельное расписание** тренировок
- 📈 **История веса** с динамикой
- 💾 **Экспорт/импорт** данных (JSON)
- 🔄 **Работает офлайн**

## 🛠️ Технологии

- HTML5 + CSS3 + Vanilla JS
- localStorage для данных
- Service Worker для офлайна
- GitHub Pages для хостинга
